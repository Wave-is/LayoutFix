# LayoutFix

LayoutFix — это современная, минималистичная и полностью локальная замена Punto Switcher.
Она перехватывает нажатия горячих клавиш и автоматически исправляет раскладку или регистр введенного текста. 

В отличие от многих аналогов, LayoutFix:
1. **Не требует подключения к сети.**
2. Работает через глобальные хуки клавиатуры (не использует сторонние тяжелые библиотеки).
3. Построен на современной архитектуре (.NET 8, Clean Architecture).

## Особенности
- Автоматическое испределение последней набранной абракадабры (например, `ghbdtn` -> `привет`).
- Изменение регистра текста (циклически: `hello` -> `Hello` -> `HELLO`).
- Транслитерация (`привет` -> `privet`).
- Перевод чисел в текст (например, `123` -> `сто двадцать три`).
- Легкий системный трей-интерфейс.

## Использование
Запустите `LayoutFix.exe`.
Приложение появится в системном трее (правый нижний угол экрана, возле часов).

**Горячие клавиши по умолчанию:**
- `Pause` (или `F12` в зависимости от настроек) — исправить раскладку выделенного текста (или последнего слова, если ничего не выделено).
- `Shift+Pause` — изменить регистр (Change Case).
- `Alt+Scroll` — транслитерация.
- `Alt+N` — перевести число в текст.

Кликните правой кнопкой мыши по иконке в трее и выберите **Settings**, чтобы изменить горячие клавиши.

## Требования
- ОС: Windows 10/11
- .NET 8.0 Desktop Runtime

## Сборка из исходников
Для сборки проекта убедитесь, что у вас установлен .NET 8 SDK.
Выполните:
```cmd
dotnet build src/LayoutFix/LayoutFix.csproj -c Release
```
