## 🇺🇸 English

# LayoutFix 🛠️

A lightweight, powerful Windows background utility designed to solve common typing annoyances. **LayoutFix** automatically corrects incorrectly typed text when you forget to switch your keyboard layout (say goodbye to retyping `vfibyf` into `машина`!). It also features hotkeys for instant text case conversion and **Offline AI Translation** using local LLMs right inside any window.

### Core Features:
- **Zero-Configuration Magic**: Automatically detects installed Windows keyboard layouts.
- **Smart Auto-Correction**: If what you typed is a valid word in the *current* layout, LayoutFix will *not* auto-convert it, preventing false positives.
- **Offline AI Translator**: Highlight text in any app, press a hotkey, and get an instant translation! Supports downloading local LLMs (Qwen, ALMA, Gemma) to translate without internet!
- **Text Manipulation Hotkeys**: Instantly switch the layout of highlighted text, or toggle its case (UPPERCASE, lowercase, Title Case) with a hotkey.
- **App Blacklist**: Add specific games or IDEs (like Dota 2 or Visual Studio) to exceptions so the app never interferes with your workflows.
- **Custom Dictionary Exceptions**: Whitelist specific abbreviations or slang.
- **Native UI**: Beautiful Dark/Light theme support matching your Windows 11 aesthetics.

### Installation
1. Go to the [Releases](../../releases) tab.
2. Download `LayoutFix_Setup.exe`.
3. Install the app. It will silently run in your system tray.

## 🇷🇺 Русский

# LayoutFix 🛠️

**LayoutFix** — это умная и быстрая утилита для Windows, которая автоматически исправляет опечатки, когда вы забыли переключить язык (например, `ghbdtn` -> `привет`). Кроме того, программа умеет переводить выделенный текст с помощью **локальных нейросетей без интернета** и менять регистр текста по нажатию одной кнопки!

### Основные возможности:
- **Магия без настроек**: Программа автоматически считывает ваши установленные раскладки.
- **Золотое правило**: Если набранный текст является существующим словом в *текущей* раскладке, программа его не тронет! Это избавляет от ложных срабатываний при наборе аббревиатур.
- **Оффлайн ИИ-Переводчик**: Выделите текст в любом окне и нажмите хоткей! Программа сама определит язык и переведет текст. Поддерживается загрузка локальных моделей (Qwen, ALMA), которые работают полностью оффлайн на вашей видеокарте или процессоре.
- **Хоткеи для текста**: Выделите абракадабру и смените ее раскладку одним нажатием. Или поменяйте регистр (ЗАГЛАВНЫЕ, строчные).
- **Исключения программ**: Добавьте игры (Dota 2) или рабочие среды (Visual Studio) в черный список, чтобы LayoutFix не мешал вам играть или писать код.
- **Словарь исключений**: Добавляйте свои специфические слова в белый список.
- **Нативный интерфейс**: Красивая темная и светлая тема под стиль Windows 11.

### Установка
1. Перейдите на вкладку [Releases](../../releases).
2. Скачайте `LayoutFix_Setup.exe`.
3. Установите программу — она свернется в системный трей и будет работать в фоне.

# LayoutFix 🛠️

**LayoutFix** — це розумна та потужна утиліта для Windows, яка автоматично виправляє помилки, коли ви забули перемикнути мову (наприклад, `ghbdtn` -> `привіт`). Програма також вміє перекладати виділений текст за допомогою **локальних нейромереж без інтернету** та змінювати регістр тексту гарячими клавішами!

### Основні можливості:
- **Магія без налаштувань**: Програма автоматично адаптується під ваші розкладки.
- **Золоте правило**: Якщо набраний текст є існуючим словом у *поточній* розкладці, програма його не чіпатиме! Ніяких хибних спрацьовувань.
- **Офлайн ШІ-Перекладач**: Виділіть текст у будь-якому вікні та натисніть хоткей! Програма сама визначить мову та перекладе текст. Підтримується завантаження локальних моделей (Qwen, ALMA), що працюють повністю офлайн.
- **Гарячі клавіші для тексту**: Виділіть текст і змініть його розкладку або регістр (ВЕЛИКІ, малі літери) в один клік.
- **Винятки програм**: Додайте ігри (Dota 2) або IDE (Visual Studio) у чорний список, щоб програма не заважала.
- **Словник винятків**: Додавайте власні специфічні слова.
- **Сучасний дизайн**: Красива темна та світла тема в стилі Windows 11.

### Встановлення
1. Перейдіть на вкладку [Releases](../../releases).
2. Завантажте `LayoutFix_Setup.exe`.
3. Встановіть програму — вона згорнеться в системний трей.
